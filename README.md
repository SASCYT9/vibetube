# 🎵 VibeTube — YouTube & SoundCloud Плеєр з Еквалайзером та MPRIS

**VibeTube** — це сучасний, високопродуктивний десктопний аудіоплеєр, побудований на базі **Electron** (фронтенд) та **Python** (бекенд). Інтерфейс плеєра виконаний у преміальному стилі Spotify із темною матовою темою та яскравими зеленими неоновими акцентами, а також повністю інтегрований у систему Linux через протокол MPRIS.

---

## ✨ Основні особливості (Features)

* **🎨 Дизайн у стилі Spotify:** Сучасний темний інтерфейс із квадратними обкладинками треків, плавними переходами, ефектами наведення (hover overlays) та лаконічною ергономікою.
* **🧠 Розумний аналіз контенту:** Інтегровані алгоритми фільтрації YouTube Shorts. Додаток автоматично очищає плейлисти від коротких кліпів (менше 60 секунд) та відео з URL-адресами `/shorts/`, залишаючи лише повноцінні музичні композиції.
* **🎛️ Графічний Еквалайзер (у модальному вікні):**
  * 7-смуговий точний еквалайзер (від 60 Гц до 15 кГц).
  * Попередньо встановлені аудіопресети (`Flat`, `Bass Boost`, `Vocal Boost`, `Pop`, `Rock`, `Jazz`, `Electronic`).
  * Спеціальні модифікатори звуку: регульований *Bass Boost* (+dB) та *Реверберація* (Reverb).
  * Швидкі пресети ефектів: **Nightcore** (прискорення, збереження тональності) та **Slowed & Reverb** (уповільнений темп із глибоким відлунням).
* **📊 7 динамічних Canvas-візуалізаторів:**
  * Режими: *Смуги (Bars)*, *Хвиля (Wave)*, *Шпильки (Spikes)*, *Спектр (WMP Spectrum)*, *Децибели (Decibels)*, *Concentric Rings (Battery)*, *Космічний пил (Particle)*.
  * **Бас-реактивне світіння:** Усі візуалізатори та обкладинка треку в реальному часі пульсують і масштабуються в такт низьким частотам (басу).
* **🖥️ Глибока інтеграція з Linux (MPRIS):**
  * Повна синхронізація з системними пунктами управління медіа (GNOME Media Widget, KDE Plasma Tray, медіа-клавіші клавіатури).
  * **Локальне кешування обкладинок:** Сервер автоматично скачує прев'ю треку з YouTube/SoundCloud, зберігає локально і передає системі `file://` шлях, гарантуючи відображення обкладинки у віджетах сповіщень Linux.
  * Підтримка керування гучністю, паузою, відтворенням та перемиканням треків безпосередньо з системи.
* **📂 Синхронізація з Firefox:** Можливість автоматичного зчитування історії переглядів та власних плейлистів користувача безпосередньо з локальної бази даних сесій браузера Firefox.

## 🏗️ Архітектура (Architecture)

Додаток використовує гібридну клієнт-серверну модель:
1. **Frontend (Electron):** Відображає інтерфейс HTML5/CSS3/JS, обробляє аудіо через Web Audio API для еквалайзера та візуалізації на Canvas, а також взаємодіє з бекендом через HTTP REST API.
2. **Backend (Python Server):** Працює на порті `8000`. Використовує `yt-dlp` для швидкого парсингу метаданих та потокового стрімінгу аудіо, взаємодіє з системною шиною D-Bus через `pydbus` для реалізації протоколу MPRIS.

## 🚀 Встановлення та запуск (Installation & Run)

### 📋 Попередні вимоги
Переконайтеся, що у вашій системі встановлено:
* **Node.js** (версії 16+)
* **Python 3**
* **yt-dlp** (встановлено глобально в системі)
* **Firefox** (якщо ви хочете синхронізувати свої YouTube-плейлисти)

### 🔧 Кроки для встановлення

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/yourusername/youtube-player.git
   cd youtube-player
   ```

2. Встановіть залежності Node.js:
   ```bash
   npm install
   ```

3. Переконайтеся, що встановлено системні залежності Python для D-Bus:
   ```bash
   pip install pydbus
   ```
   *(Примітка: для роботи pydbus у системі повинні бути встановлені пакети `python3-gi` або `gobject-introspection`)*

### 🎮 Запуск додатка
Запустіть плеєр за допомогою npm скрипту:
   ```bash
   npm start
   ```

## 🛠️ Технологічний стек (Tech Stack)

* **Frontend:** Vanilla JS, CSS3 Custom Properties, Web Audio API, HTML5 Canvas.
* **Desktop Wrapper:** Electron (v42+).
* **Backend:** Python 3 (BaseHTTPRequestHandler).
* **Streaming Engine:** `yt-dlp` (з підтримкою JS runtimes).
* **Linux D-Bus API:** `pydbus`, `GLib` (GObject Introspection).
